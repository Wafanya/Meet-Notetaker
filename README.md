# Meet Notetaker

Chrome-екстеншн який автоматично зберігає транскрипти Google Meet у Notion, генерує AI-самарі і доставляє у Slack DM.

## ✨ Можливості

- 📝 Зберігає транскрипт зустрічі в Notion
- 🤖 Авто AI-самарі через OpenAI
- 💬 Доставка в Slack DM
- 💾 Постійне локальне збереження кожні 10 сек — нічого не губиться при крешах/вимиканні комп'ютера
- ↺ Resume логіка: після крешу екстеншн продовжує з того ж місця
- 🔁 Retry: якщо інтернет відпав → екстеншн ретраїть webhook автоматично

## 📦 Встановлення

### 1. Завантаж код

```bash
git clone https://github.com/Wafanya/Meet-Notetaker.git
```

### 2. Встанови в Chrome

1. `chrome://extensions/`
2. Увімкни **Developer mode**
3. **Load unpacked** → обери папку

## ⚙️ Налаштування

### Notion

Тобі **НЕ** потрібно створювати свою Notion-інтеграцію — вона вже налаштована.

Тобі треба тільки:

1. **Створи власну базу** у Notion (або скористайся наданою адміном). Поля бази:
   | Назва | Тип |
   |---|---|
   | `Name` | Title |
   | `Date` | Date |
   | `Participants` | Text |

2. **Дай інтеграції доступ до бази**:
   - Відкрий базу → `•••` (вгорі справа) → **Connections**
   - Додай інтеграцію **Meet Notetaker**
   - Якщо її нема в списку — попроси адміністратора додати тебе

3. **Скопіюй Database ID** з URL:
   ```
   notion.so/myworkspace/352146034ca98057b666f23bb6ae2512?v=...
                          └──────────────┬───────────────┘
                                   Database ID
   ```

### Slack

1. Slack → клікни на свою аватарку → **Profile**
2. Натисни `•••` → **Copy member ID**
3. Скопіюється ID типу `U073SU0590U`

> Якщо це поле залишити порожнім — самарі не приходитиме в Slack. Notion-сторінка створиться як зазвичай.

### Введи дані

1. Клікни іконку екстеншну
2. Встав **Database ID** і **Slack User ID**
3. **Save**

## 🚀 Як користуватись

1. Зайди на зустріч у Google Meet
2. Captions (CC) увімкнуться автоматично
3. Усе. Запис іде у фоні.
4. При завершенні зустрічі (Leave call, закриття вкладки, або 5+ хв тиші) → транскрипт автоматично йде в Notion + AI-самарі у Slack

## 🛡 Стабільність

Все що пише екстеншн зберігається кожні 10 секунд в локальному сховищі браузера. Якщо:

- Інтернет відпав → коли повернеться, екстеншн ретраїть автоматично
- Браузер крешнув → коли запустиш знов і зайдеш на той самий Meet → продовжить з того ж місця
- Комп вирубили → дані залишаться. При наступному запуску браузера екстеншн знайде незакриту зустріч і відправить її

## 🔒 Безпека

- Жодних токенів у браузері юзера — все на сервері
- Транскрипти йдуть напряму у твою Notion-базу (через HTTPS)
- Slack-повідомлення приходять лише тобі
